Voice Typing

Privacy Policy

Voice Typing is a Chrome extension that sends audio to the ElevenLabs Scribe API only to generate a transcription for the user who started the recording.

What data the extension uses

  • Microphone audio when you choose microphone recording.
  • Tab audio when you choose tab capture.
  • Desktop audio when you choose desktop capture.
  • Your ElevenLabs API key, stored in Chrome extension storage.

How data is used

Audio is transmitted over HTTPS to ElevenLabs to create a transcript. The extension does not run its own analytics, ad tracking, or data resale pipeline.

What is stored locally

The extension stores user settings such as selected language, auto insert preference, auto copy preference, recent transcription history, and the API key needed to call ElevenLabs.

What is not collected by this repository

This open source project does not include any backend service for collecting transcripts, microphone recordings, or account data from users.

Third-party services

Transcription requests are handled by ElevenLabs. Review their terms and privacy practices before using the extension in production or with sensitive audio.
