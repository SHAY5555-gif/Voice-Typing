# Code of Conduct

## Our Pledge

We want Voice Typing to be a respectful, useful project for everyone involved.
Contributors, maintainers, and users are expected to act professionally and
constructively.

## Expected Behavior

- Be respectful and direct
- Assume good intent, but stay factual
- Focus feedback on the code and the behavior, not the person
- Share context clearly when reporting bugs or reviewing changes

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Publishing private information without permission
- Bad-faith disruption of discussions, reviews, or issues
- Repeatedly ignoring project guidelines

## Enforcement

Project maintainers may edit, hide, lock, or remove comments, issues, pull
requests, or contributions that violate this code of conduct.
